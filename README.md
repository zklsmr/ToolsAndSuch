## Usage Example
```bash
# docs are called using -h
# Summarization function example 
python3 sums.py [-h] pdf_in txt_out max_l min_l chunk_arg


positional arguments:
  pdf_in      PDF file to summarize
  txt_out     output file
  max_l       max length
  min_l       min length
  chunk_arg   chunk proportion from total text
```
